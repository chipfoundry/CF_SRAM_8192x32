# LEF Directory

This directory contains the Library Exchange Format (LEF) file for the CF_SRAM_8192x32 macro.

## Expected File

- `CF_SRAM_8192x32.lef` - LEF format file

## Generation

This file should be generated from the physical layout. It provides:
- Abstract view of the macro for place & route tools
- Pin locations and metal layers
- Blockage information
- Size and origin information

## Reference

See `CF_SRAM_4096x32/lef/CF_SRAM_4096x32.lef` for the equivalent file in the 16KB SRAM macro.
